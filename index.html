Impostazioni

Collega un GitHub Gist per sincronizzare i tuoi allenamenti su qualsiasi device. Tutto rimane nel tuo account GitHub, gratis e senza limiti.

Come ottenere il token GitHub 1. Vai su github.com → Settings → Developer settings
2. Personal access tokens → Tokens (classic)
3. Generate new token → seleziona solo scope gist
4. Copia il token e incollalo qui sotto
Salva questo link nei preferiti. Se cancelli la cronologia, riaprilo e l'app si riconfigura automaticamente.

Istruzioni GitHub Pages
1. Crea repo fitness-tracker su GitHub
2. Carica questo file come index.html
3. Settings → Pages → Branch: main → Save
4. Accedi da: username.github.io/fitness-tracker
5. Genera token GitHub con scope gist e configuralo sopra
6. Al primo salvataggio il Gist viene creato con fitness-data.json e program.json
7. Per aggiornare la scheda: modifica program.json nel Gist
